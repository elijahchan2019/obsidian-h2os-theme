# H2 OS

![H2 OS 主题预览](screenshot.png)

> 一款几何美学的 Obsidian 亮色主题。白色画布，克制的一加红，方、圆、三角各司其职。氢 OS 停更多年，但它的设计纪律不该跟着退役——H2 OS 把这套纪律翻译到今天的书写环境里。

## 一个故事：氢徽

![H2 OS 视觉系统](feature-artboard.png)

打开一个空标签页，你会看到主题的徽记——三个几何形状，一套固定关系：

- **方是结构。** 你的笔记、你的库，承载一切的底座，所以最大、最安静。
- **圆是日常。** 每一次点击、勾选、翻阅，人与库的接触，居于结构之上。
- **三角是灵感。** 最小的形状，尖端从结构与日常的交叠处**冒出来**。

结构承载，日常流转，灵感冒尖。这不是装饰画，是整个主题的工作原理图。

## 一条纪律：红色从不装饰

红色在这个主题里是**信号**，不是颜色。它只在两种时刻出现：

- **"你在这里"**——当前页签的红字、文件树里的红点、正在书写的红色光标、钉住页签的红点。
- **"刚刚发生"**——勾选任务时从圆心漾开的红、搜索命中时扫入的红、悬停链接时重描的红线。

除此之外，界面是白与灰的。所以每一次红色出现，它都在对你说话——一屏一个红元素，多了就是噪音。用久了你会发现自己开始**读**这些红点，像读标点符号一样。

## 一层重绘：没有一个记号是默认的

H2 OS 不改一个文字字形，但重绘一切非文字记号。分隔线是三段式的；引号是两个咬合的几何方块；列表点是方的；勾号、图钉、设置图标、22 种任务状态标记——全部从方、圆、三角三个原子重新生成。

这些东西你每天看几百次却从不注意。把它们全部换掉之后，界面说不出哪里变了，但处处都变了。

## 一张画布：不靠边框建立秩序

窗口是一整张连续的白色画布，两侧工具区用一档极轻的色温差安静退后。没有卡片、没有成排的分隔线——全部结构线收敛到**两条红线**：

- 一条横线沿导航层下沿贯穿整个窗口。打开应用时，它从左向右刷过——先缓慢蓄力，越过中点后一气收尾。
- 一条竖线立在设置页左栏。打开设置时，它自上而下落定。

每天开机第一眼，主题向你签到一次。

## 有分寸的动效

纪律管常态，愉悦住事件。界面平时是静的，只在你的动作值得回应的瞬间动一下：勾选的墨滴、氢徽的两拍入场、搜索命中的轻扫、折叠展开的上浮。全部尊重 `prefers-reduced-motion`；移动端只保留直接操作反馈。

## 推荐用法

- **开启内嵌标题**（设置 → 外观 → 显示内嵌标题）：每篇笔记的排面级开场和三段线开幕动效都长在它上面。
- **收起左侧功能区**：主题为极简而生，隐藏 ribbon 后画布更完整——高频入口都有快捷键。
- 保持亮色模式。白色画布与灰红层级是作为一个整体调校的；切到暗色时 Obsidian 会用原生暗色兜底，主题不提供仓促反相的配色。

## 推荐字体

主题不内置字体文件。Poppins 或 Montserrat 呈现拉丁文的几何字形，中文优先思源黑体。macOS 用户可用 Homebrew 安装：

```bash
brew install --cask font-poppins font-montserrat
```

未安装时主题回退到平台自带的近似几何字体（macOS 的 Avenir Next、Windows 的 Segoe UI Variable），再回退系统无衬线。

## 安装

从 Obsidian 社区主题市场安装：

1. 打开 **设置 → 外观 → 主题 → 管理**
2. 搜索 **H2 OS**
3. 安装并启用

手动安装：从最新 release 下载 `theme.css` 与 `manifest.json`，放入：

```text
<vault>/.obsidian/themes/H2 OS/
```

## 兼容性

- Obsidian 1.5.0+
- 亮色模式 · 桌面与移动端 · 打印与 PDF · 无需插件
- 主题使用原生 CSS Nesting。若部分样式未生效，请下载最新版 Obsidian 安装包以更新内置 Chromium。
